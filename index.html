Smart Blind Stick Using IoT:

The Smart Blind Stick using IoT is an assistive device designed to help visually impaired people navigate safely. It uses ultrasonic sensors and an Arduino microcontroller to detect obstacles in front of the user. When an object is detected, the system alerts the user through a buzzer or vibration, providing a low-cost and reliable mobility solution.
